## 📱 Responsividade

### Desktop (> 1024px)
- Layout de 3 colunas para features
- Hero com imagem à direita
- Menu horizontal completo

### Tablet (768px - 1024px)
- Layout de 2 colunas
- Hero empilhado
- Menu colapsável

### Mobile (< 768px)
- Layout de 1 coluna
- CTA fixo no rodapé
- Menu hambúrguer
- Imagens otimizadas (WebP)
